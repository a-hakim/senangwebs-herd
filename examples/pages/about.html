SenangWebs Herd (SWH) is a lightweight web library for managing multiple HTML files within a single page.

What is SWH?

SWH embeds external HTML files inside iframe elements and allows users to open, close, and switch between tabs. The open tabs and active tab state are persisted using localStorage, allowing the layout to remain consistent after page refresh.
